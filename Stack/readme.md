
# Stack 

A Stack follows **LIFO** (Last In, First Out) order — the last element added is the first one removed.
Think of a stack of plates: you add and remove from the top only.

## Implementations
- **Array-based Stack** — uses Python list internally
- **Linked List based Stack** — uses custom Node structure

## Operations
- `push(item)` — add item to top
- `pop()` — remove item from top
- `peek()` — view top item without removing
- `is_empty()` — check if stack is empty

## Time Complexity
| Operation | Time |
|-----------|------|
| Push | O(1) |
| Pop | O(1) |
| Peek | O(1) |
